# UTD (University of Texas at Dallas) Top Trader Contest

## Overview

This project implements a volatility-adjusted momentum trading strategy for the [UTD Top Trader Contest](https://jindal.utdallas.edu/top-trader-2/). The strategy selects the top N stocks based on a composite score that combines multiple momentum and quality metrics, then holds them for a specified period.

## Windows 11 Setup

1. Create virtual environment

```bash
python -m venv venv
```

2. Activate virtual environment

```bash
.\venv\Scripts\Activate.ps1
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Verify installation

```bash
pip list
```

5. Run Jupyter notebook

```bash
jupyter notebook
```
